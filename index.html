ℹ️ Tentang Aplikasi Ini

Aplikasi Dashboard Indikator Strategis BPS Kota Pekalongan dikembangkan untuk menyediakan akses cepat, visualisasi interaktif, dan detail metadata dari data-data kunci yang diterbitkan oleh Badan Pusat Statistik.

🎯 Tujuan Utama

Memfasilitasi pengambilan keputusan yang lebih cepat dan berbasis data bagi pengguna internal dan eksternal.

💾 Sumber Data

Data diolah secara real-time dari Google Sheets multi-sheet, memastikan informasi selalu up-to-date.

🛠 Teknologi

Dibangun dengan HTML5, CSS3 Modern, Google Charts API, dan JavaScript ES6+ untuk performa optimal.

✨ Fitur Unggulan

  • ✓ Visualisasi data interaktif dengan grafik tren
  • ✓ Filter dinamis berdasarkan arah perubahan
  • ✓ Metadata lengkap setiap indikator
  • ✓ Responsive design untuk semua perangkat
